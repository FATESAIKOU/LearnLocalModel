# Decision 001 — 使用 Gemma 4

## 背景

原始任務是部署 Gemma 模型，之後使用者明確指定「想用 Gemma 4」。

## Input

| 項目 | 內容 |
|---|---|
| 使用者要求 | 想用 Gemma 4 |
| 既有前提 | Ubuntu Server / RTX 2070 SUPER 8GB / 64GB RAM |

## DA 表

| 欄位 | 內容 |
|---|---|
| 技術/決策的採用前提 | 需以使用者指定的模型世代為主 |
| 採用目的(效果) | 鎖定後續評估對象，避免方案比較建立在錯誤模型版本上 |
| 結果影響內容 | 後續模型大小、部署工具、硬體適配都需以 Gemma 4 重新評估 |
| 結果影響範圍 | 全部部署流程 |
| 採用成本 | 低 |

## 結論

改以 **Gemma 4** 為唯一部署目標，不再沿用先前對舊版 Gemma 的假設。
